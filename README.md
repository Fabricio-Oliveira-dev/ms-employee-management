# About this project
This project is an employee management web application using microservices with Kafka and RabbitMQ as event-driven which is being built yet.

# Technologies used
### Back End
- Java
- Spring Boot
- Spring Data JPA
- Spring Cloud
- Zipkin
- Resilience4J
- Kafka
- RabbitMQ
- MySQL

### Front End
- JSX
- React
